# 示例内容

## 1. 图片生成提示词示例 (用于AI绘画工具)

**示例：如何画一只可爱的小兔子**

```
生成一张儿童简笔画风格的四格教学图片，画面简洁，线条流畅，适合儿童学习绘画。整体图片比例为3:4。
第一格：画一个简单的椭圆形作为兔子的身体。
第二格：在身体上方添加一个圆形作为头部，并画出两只长长的耳朵。
第三格：画出兔子的四条腿和毛茸茸的短尾巴。
第四格：画上眼睛、鼻子、嘴巴和胡须，完成一只可爱的小兔子。
```
