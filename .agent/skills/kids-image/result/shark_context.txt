# 凶猛的鲨鱼简笔画生成结果

## 1. 图片生成提示词 (已生成)

**提示词：**
```
生成一张儿童简笔画风格的四格教学图片，画面简洁，线条流畅，适合儿童学习绘画。整体图片比例为3:4。
第一格：画一个简单的长椭圆形作为鲨鱼的身体。
第二格：在背部中央添加一个大的三角形背鳍，并在后部画出新月形的尾鳍。
第三格：在侧面画出胸鳍，并点上两个圆圆的大眼睛。
第四格：画出一个张开的大嘴巴，露出一排锯齿状的“凶猛”牙齿，并加上三道鳃裂。
```

```

1. **简单易上手：** 只要四步，从一个椭圆变身成大鲨鱼，零基础的小朋友也能轻松掌握！

🎨 **画法步骤：**
1️⃣ 第一步：先画一个长长的椭圆作为身体。

3️⃣ 第三步：两边画上鳍，还有亮晶晶的大眼睛。
```
