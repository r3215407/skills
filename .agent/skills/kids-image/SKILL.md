# 儿童简笔画四格教学图片生成技能

## 概述
此技能旨在生成一张包含四格步骤的儿童简笔画教学图片，并提供配套的小红书推广标题和文案，帮助用户轻松创建亲子绘画教程内容。

## 1. 图片生成提示词 (用于AI绘画工具)

**通用结构：**
```
生成一张儿童简笔画风格的四格教学图片，画面简洁，线条流畅，并且使用不同的颜色， 背景需要是一张真实的白纸。整体图片比例为3:4。
细腻纸张纹理背景，浅色石膏墙颗粒质感，柔和自然光，马卡龙配色，低饱和温暖色调，手绘卡通线条，线条柔软不生硬，轻微水彩晕染效果，画面干净留白，柔和阴影，整体温馨治愈氛围，高清细节.
第一格：[绘画步骤1的详细描述]
第二格：[绘画步骤2的详细描述]
第三格：[绘画步骤3的详细描述]
第四格：[绘画步骤4的详细描述]
```
**示例：** 请参考 `result/context.txt` 中的 "图片生成提示词示例"。

## 4. 图像生成与保存

此技能通过AI绘画工具生成四格教学图片和上色后的最终图片。

**生成步骤图 (result/step.png):**
使用上述 "图片生成提示词" (例如 `result/context.txt` 中的小兔子示例) 提交给AI绘画工具，生成一张包含四格步骤的简笔画教学图片。

**生成上色图 (result/color.png):**
在生成步骤图后，提取第四格的简笔画内容，并使用AI绘画工具对其进行上色，生成一张彩色的最终图片。

**注意:** 本AI无法直接生成图片，已为您创建了 `result/step.png` 和 `result/color.png` 的占位文件。
